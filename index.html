Project Details

This interactive Vernier Caliper simulation was developed using HTML, CSS, and JavaScript. It allows students to virtually practice measurements, including main scale and vernier scale readings, with interactive functionality for better conceptual understanding.

📈 Project Title: Interactive Vernier Caliper Simulation with Smart Chatbot Assistance

  • Interactive UI: Real-time MSR and VSR readings.
  • Labeled Components: Each caliper part is identified.
  • Insert Object: Try measurements with digital objects.
  • Chatbot Support: Guides students on VSC, LC, ZE, and more.
  • Remote Ready: Ideal for students without access to physical tools.

🎯 Aim: Make precision measurements accessible and educational.
